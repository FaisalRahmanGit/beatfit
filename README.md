# BeatFit

Lightweight static landing site for a fitness/music web project. This repo contains the frontend assets (HTML, CSS, JS and images) for the BeatFit demo site.

Status: Prototype / static site

Features

- Minimal single-page layout (`index.html`)
- Custom styles in `assets/css/style.css`
- Small JS interaction in `assets/js/custom.js`
- Image assets in `assets/images/`

Quick start

- Open the site locally by double-clicking `index.html` or serve it with a static server.

Project structure

- `index.html` — main entry
- `assets/css/style.css` — styles
- `assets/js/custom.js` — scripts
- `assets/images/` — images and icons
